# Graph Semantic Patterns

Supported patterns:
- hit
- heavy_hit
- crit
- damage_taken
- death
- pickup
- objective_progress
- objective_complete
- low_health
- danger
- wave_start
- wave_end
- run_success
- run_fail
- suppressed
- detected
- order_confirmed
- hostage_secured

Rule: one primary A/V pattern and at most one enhancer per semantic event.
