# xLabMcp Tools by Step Archive

## Policy snapshot

Before each implementation step, the stack was expected to run a preflight check, verify editor readiness, execute the requested action, inspect the console, and confirm the target artifact exists.

## Step-to-capability snapshot

1. Structure folders:
- project root selection, editor readiness, asset folder creation, asset existence check, console inspection

2. Create the rescue scene:
- project root selection, editor readiness, scene creation, scene save, asset existence check, console inspection

3. Create the bootstrap scene:
- project root selection, editor readiness, scene creation, scene save, asset existence check, console inspection

4. Apartment layout placeholders:
- scene open, object creation, object modification, scene save, console inspection, scene capture

5. Add mission director object:
- scene open, object creation, component add/set, scene save, console inspection

6. Implement mission state service:
- script generation/editing, compile status, console inspection, asset refresh

7. Implement objective service:
- script generation/editing, compile status, console inspection

8. Add operative prefabs and activation switch:
- prefab creation/open/save/instantiate, component add, scene open/save, console inspection

9. Add movement commands:
- script generation/editing, graph create/connect, compile status, console inspection

10. Add attack-target command:
- script generation/editing, graph editing, compile status, console inspection

11. Combat baseline:
- script generation/editing, component add, scene save, compile status, console inspection

12. Friendly fire and tests:
- script generation/editing, edit-mode tests, test result inspection, console inspection

13. Enemy prefab and movement:
- prefab creation, component add, prefab save, scene open/instantiate/save

14. Vision cone:
- script generation/editing, component set, scene save, console inspection, scene capture

15. Gunshot noise pathway:
- script generation/editing, component add, compile status, console inspection

16. Alert FSM:
- graph editing, script generation/editing, scene save, console inspection

17. Hostage free/follow:
- prefab generation/editing, script generation/editing, scene save, console inspection

18. Extraction and resolver:
- object creation, component add, script generation/editing, scene save, console inspection

19. Result UI and localization keys:
- UI generation/editing, localization key add, scene save, console inspection

20. Save schema and autosave:

21. Load and restore:

22. Full verification:

23. Final cleanup:
- modified asset listing, change summary, documentation update, console inspection
